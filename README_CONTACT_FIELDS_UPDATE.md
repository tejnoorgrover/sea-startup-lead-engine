# Contact Fields Update

This update adds founder/contact fields to the live dashboard:

- `founder_name`
- `founder_title`
- `business_email`
- `business_phone`
- `contact_enrichment_status`

The dashboard now shows these fields in the lead table, sales-call brief, CSV export, and compliance section.

Important: live public news/RSS sources usually do not contain verified founder emails or phone numbers. The app therefore does not guess or fake personal data. Email and phone fields are displayed as `Not publicly listed in source` until enriched through compliant sources such as official company websites, licensed enrichment APIs, or CRM-approved enrichment tools.
